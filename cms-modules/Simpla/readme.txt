Модуль оплаты Payler для Simpla CMS.

1. Установка модуля
Распаковать архив и поместить папку Payler в каталог с другими платежными модулями - <каталог сайта>/payment

2. Настройка модуля
Открыть настройки сайта. На вкладке "Оплата" нажать на "Добавить новый способ оплаты". 

- Ввести название, как это будет видеть клиент (например, "Оплата через Payler"), установить флаг "активен".
- В выпадающем списке выбрать "Payler", валюта - "Рубли" (модуль рассчитан только на оплату в рублях). 
- Указать платежный ключ (выдается при создании аккаунта в Пэйлер).
- Указать тип создаваемых сессий (при одностадийном деньги сразу списываются с карты, при двухстадийном - только блокируются, затем менеджеру надо выполнить списание через Личный кабинет Пэйлер по платежу)
        ВНИМАНИЕ! Вне зависимости от выбора типа платежа, в Simpla заказ будет помечаться, как успешный, при блокировке ИЛИ списании средств.
- Указать, используется ли тестовый режим или боевой (тестовый и боевой ключи выдаются Пэйлер при создании аккаунта, на тестовом работает прием только специально выданных карт, но списание товара в магазине производится).
- Указать возможные способы доставки, для которых способ оплаты будет применим.

3. Настройка аккаунта Пэйлер
В Личном кабинете Пэйлер нажать в правом верхнем углу по email-у. В выпадающем списке выбрать "Настройки". Указать URL возврата в магазин:
http://<название вашего сайта>/payment/Payler/callback.php?order_id={order_id}
